ที่มาและความสำคัญของโครงงาน

โปรเจคนี้เป็นโปรเจคที่ได้พัฒนาจากหุ่นยนต์ที่รุ่นพี่ทำไว้ให้ควบคุมโดยใช้ Arduino โดยเราจะเขียน โค้ดรับค่าจากปุ่มแล้วให้ Arduino ไปสั่งมอเตอร์เพื่อเคลื่อนที่ ซึ่งเพื่อนำไปเป็นสื่อการเรียนการสอน ในการฝึกเขียนโปรแกรมควบคุมมอเตอร์ผ่าน Arduino ได้

2.Project - ล็อกเกอร์ปลดล็อกด้วยระบบสแกนนิ้ว

Project : ล็อกเกอร์ปลดล็อกด้วยระบบสแกนนิ้ว

Period : 2 Weeks

ที่มาและความสำคัญของโครงงาน

ในยุคที่โจรชุกชุมในบ้านเรา ทำให้อาจจะเกิดของหายได้ กลุ่มของพวกเราจึงได้ สร้างล็อกเกอร์ปลดล็อกด้วยระบบสแกนนิ้ว เพื่อความปลอดภัย โดยการที่ใช้ระบบสแกนนิ้วนอกจากจะปลอดภัยแล้ว ยังสะดวก

หลักการทำงาน

ขั้นแรกเราต้องทำการเก็บลายนิ้วมือกันก่อน เมื่อบอร์ดจำลายนิ้วมือเราได้แล้วก็ถือว่าใช้งานได้แล้ว โดยเมื่อเราสแกนนิ้ว แล้วลายนิ้วมือตรงกับที่เก็บไว้ Arduino จะสั่งจ่ายไฟให้กับ Relay เพื่อให้ Solenoid ทำงาน ทำให้ล็อกเกอร์ปลด ล็อก เมื่อลายนิ้วมือไม่ตรงกับที่เก็บไว้ Solenoid จะไม่ทำงานและล็อกเกอร์ก็จะไม่สามารถเปิดได้

3. Project - เครื่องรักษาอุณหภูมิของเหลวอัตโนมัติ

Project : เครื่องรักษาอุณหภูมิของเหลวอัตโนมัติ

Period : 1 Week

ที่มาและความสำคัญของโครงงาน

ในยุคที่นํ้ามันมีความสำคัญต่อชีวิตประจำวันอย่างมาก การเดินทางก็ต้องใช้นํ้ามัน ทำอาหารก็ต้องใช้นํ้ามัน โดยวิชาเซ็นเซอร์อาจารย์ให้ทำโครงงานโดยให้ใช้เซ็นเซอร์ในการแก้ปัญหาในอุตสาหกรรม โดยกลุ่มของพวกเราก็ได้เล็งเห็นถึงข้อเสียของนํํ้ามันก๊าสตรงที่ว่า เมื่ออุณหภูมิลดน้อยลงจะทำให้นํ้ามันก๊าสแข็งตัว และไม่สามารถใช้งานได้ โดยเราได้เล็งเห็นถึงปัญหาตรงนี้และได้คิดวิธีการแก้ไขปัญหานี้โดยการ สร้างเครื่องรักษา อุณหภูมิอัตโนมัติซึ่งจะสามารถทำให้สามารถเก็บนํ้ามันได้โดยไม่เกิดการแข็งตัว

หลักการทำงาน

เมื่อตัวถังมีอุณหภูมิที่ลดลง เซ็นเซอร์จะส่งสัญญาณไปที่ Arduino เพื่อใหให้ Arduino สั่งงานใน Heater ทำงานซึ่งเมื่อ Heater ทำงานไปช่วงนึงจะอุณหภูมิของตัวถังสูงขึ้นในระดับที่พอจะทำให้นํ้ามัน ไม่แข็งตัว เซ็นเซอร์จะส่งสัญญาณให้ Arduino ว่าอุณหภูมิถึงจุดที่นํ้ามันไม่แข็งตัวแล้ว Arduino จะสั่ง ให้ Heater หยุดทำงานเพื่อจะได้ประหยัดไฟหากจะต้องเปิด Heater ตลอดทั้งวันอาจจะกินไฟได้

เมื่อเซ็นเซอร์ตรวจจับว่าอุณหภูมิภายนอกลดลงหรืออากาศเย็นขึ้น เซ็นเซอร์จะส่งสัญญาณ Digital ไปที่่ Arduino เพื่อให้จ่ายไฟให้รีเลย์เพื่อให้ไฟเข้า Heater จึงทำให้หม้อกลับมามีอุณหภูมิที่เหมาะสม
